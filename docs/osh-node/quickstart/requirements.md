In order to build and run an **OpenSensorHub** node, you must have the following installed:

- Java 17+
- Gradle 7.3+
- Git

**OpenSensorHub** is widely available for the following platforms:

- **Supported Platforms:**
    - MacOS
    - Windows
    - Linux
    - RaspberryPi
    - Most ARM boards (with JVM)
